How PULSE Uses These Algorithms

A* -- Real-time EV Routing

Primary algorithm for ambulance navigation. Uses GPS coordinates as heuristic to find the fastest route to hospital. Re-runs every few seconds as the vehicle moves. Explores fewer nodes than Dijkstra by focusing toward the target.

Floyd-Warshall -- City Corridor Matrix

Pre-computes all-pairs shortest paths between major intersections at startup. Traffic Authority uses this matrix for instant green corridor planning with no computation delay when seconds matter.

2025 Breakthrough -- The Frontier

Breaks the O(n log n) sorting barrier by hybridizing Dijkstra + Bellman-Ford with divide-and-conquer and FindPivots. Achieves O(m log^(2/3) n). Proves faster SSSP is possible for city-scale directed road networks.

Bellman-Ford -- Graph Validator

Used offline to verify that our traffic graph has no negative cycles (which would indicate a modeling bug in edge weights). Its ability to detect negative cycles makes it a safety net, not a primary router.
